1.Types in a function
    1.1 While creating a function(destructor receive):
        const demoFunction = ({props, data}:{props:*type*, data:*type*}) => {}
    1.2 While calling a function(single argument):
        demoFunction<*type*>(argument||null)
    1.3 While calling a function(multiple argument):
        demoFunction<{argument:*type*}>(argument || null)
    special case
    1.4 While using a ref
        useRef<HTML*nameOfElement*Element>() // it means the particular Element as type
    1.5 While type can be anything that React can render eg: children
        React.ReactNode
    1.6 Event Props 
        const handleAnything=(e:React.SyntheticEvent<HTML*nameOfElement*Element>) =>{} & use e.currentTarget instead of e.target
